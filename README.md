# Coridon's GitHub Pages Site

This repository is a minimal, single-page template customized for **Coridon**.

## How to publish
1. Create a repository named exactly: `coridon.github.io`
2. Copy `index.html`, `styles.css`, `hero-illustration.svg`, `favicon.svg`, and `README.md` to the repo root.
3. Commit & push to GitHub.
4. Visit `https://coridon.github.io` (it may take a minute).

## Optional
- Add a `CNAME` file with your custom domain.
- Use Jekyll or a static site generator if you want a blog (/blog) with Markdown posts.
- Use Actions to auto-deploy from a different branch.
